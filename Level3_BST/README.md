# Binary Search Tree (BST) Implementation

## Description
This project implements a Binary Search Tree in Java with core operations.

## Features
- Insert nodes
- In-order traversal (sorted output)
- Search element in BST

## Concepts Used
- Data Structures (BST)
- Recursion
- Tree Traversal
- Java OOP
