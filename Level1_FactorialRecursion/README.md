# Factorial Calculator (Recursion)

## Description
This project calculates factorial using recursion in Java.

## Features
- Recursive factorial computation
- Handles negative numbers
- Displays factorial expansion
- Input validation

## Concepts Used
- Recursion
- Object-Oriented Programming
- Exception Handling
- User Input Handling

# Library Management System (Java)

## Description
This is a console-based Library Management System that allows users to manage books.

## Features
- Add books
- View books
- Borrow books
- Return books

## Concepts Used
- Object-Oriented Programming
- ArrayList
- CRUD operations
- Java Core Concepts
