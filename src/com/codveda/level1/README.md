# Number Guessing Game (Java Console Application)

## Description
This project is part of my Java Development Internship at Codveda Technologies.

It is a console-based number guessing game built using Object-Oriented Programming concepts.

## Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Limited attempts system
- Score calculation
- Replay option
- Input validation for incorrect entries

## Concepts Used
- OOP (Object-Oriented Programming)
- Random class
- Loops and conditionals
- Input validation
- Clean method structuring
